    3. Basic Functions

    Wat heb ik vandaag geleerd?

  • Een “function” is een herbruikbaar stuk code dat een bepaalde logica uitvoert en je een resultaat geeft.
  • Met het keyword “return” kan de functie het resultaat van de berekening geven.
  • Het keyword “return” kan alleen binnen een functie worden gebruikt.
  • Om een “function” aan te roepen, moet je beginnen met de naam, dan haakjes openen (, dan de waarde die je wilt geven aan de parameter, en dan een sluitend haakje ).

    • 15/12/2024

    4. Basic Functions II

    Wat heb ik vandaag geleerd?

  • “Functions” kunnen meerdere parameters hebben, ze moeten worden gescheiden door een komma (en een spatie voor de leesbaarheid)
  • Het argument is een waarde die wordt doorgegeven aan een "function". Het argument wordt de parameter voor die specifieke functieaanroep.
  • De parameters van een functie zijn alleen toegankelijk binnen die functie. (Dit concept wordt variabele reikwijdte genoemd)

    6. Strings

    Wat heb ik vandaag geleerd?

  • Een string is een tekst, een verzameling tekens die je kunt schrijven om een naam, stad, land, beschrijving, producttitel, enz. weer te geven.
  • Strings moeten beginnen met een dubbel aanhalingsteken “ en eindigen met een.
  • Het vergeten van de dubbele aanhalingstekens rond strings is de meest voorkomende fout die beginners maken.
  • .length is een eigenschap van strings die het aantal tekens in een string weergeeft.

    7. Console log

    Wat heb ik vandaag geleerd?

  • Met console.log() kun je je code visualiseren in de console.
  • De console is alleen zichtbaar voor jou (de ontwikkelaar), maar niet voor de eindgebruiker.
  • console.log() helpt je bugs te vinden "debugging".
  • .console.log vervangt return NIET
  • console.log zal nooit het resultaat van een functie wijzigen; het zal je alleen een string of een getal in de console laten zien.
  • Sample Usage is bedoeld om voorbeelden te laten zien van hoe je functions worden aangeroepen.
  • Pas op voor hardcoding - het voorbeeldgebruik is alleen bedoeld als voorbeeld.

    • 14/1/2025

    8. Strings II

    Wat heb ik vandaag geleerd?

  • De methode .toUpperCase() maakt een nieuwe kopie van een tekenreeks en converteert elk teken naar hoofdletters.
  • De methode .toLowerCase() maakt een kopie van de tekenreeks met allemaal kleine letters.
  • Let op de manier waarop deze methoden worden aangeroepen: .toUpperCase() met een hoofdletter U en een hoofdletter C.
  • Hetzelfde geldt voor .toLowerCase() met een hoofdletter L en een hoofdletter C.
  • De () aan het einde van de methode zijn verplicht.
